赛博朋克毛玻璃风格

欢迎来到未来！这个升级为您的博客带来了充满科技感的赛博朋克风格， 包括毛玻璃效果、霓虹灯、动态背景和交互特效。

🎨 主要特效

毛玻璃效果

所有卡片和容器都采用毛玻璃背景，营造未来感

backdrop-filter: blur(20px);

霓虹灯效果

文字和边框都有霓虹发光效果

text-shadow: 0 0 10px #00ffff;

动态背景

多层渐变背景和移动网格线

animation: cyberpunk-bg 20s;

交互特效

鼠标跟随、点击烟花、键盘涟漪

fireworks: enable: true;

🎨 赛博朋克色彩方案

主色调 - 霓虹蓝

#00ffff - 青色，用于主要文字和边框

辅助色 - 霓虹粉

#ff00ff - 洋红色，用于分类和标签

强调色 - 霓虹黄

#ffff00 - 黄色，用于重要元素

深色背景

#0a0a0a - 深灰色，营造赛博朋克氛围

📖 使用说明

如何应用这些效果到您的博客：

# 1. 生成静态文件 hexo clean && hexo generate # 2. 本地预览 hexo server # 3. 部署到GitHub Pages hexo deploy

• 毛玻璃赛博朋克样式文件
• 动态特效JavaScript脚本
• 优化的主题配置
• 响应式设计支持
